stopPropagation() is used to stop an event bubbling up. The equivalent in IE8 is the cancelBubble property.

Computed style

IE8 had a proprietary currentStyle property which is almost equivalent to getComputedStyle() method. In some cases, currentStyle property may return a style with NaN or 'auto', etc. as its value.
